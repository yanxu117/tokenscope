30 秒上手

克隆、解析数据、打开仪表盘。

# 克隆
git clone https://github.com/Singu-larity-ai/tokenscope.git
cd tokenscope && npm install

# 解析你的数据
npm run parse

# 构建并打开
npm run build && npm run preview
